# 데이터 모델 명세 (v3)

## 시간 단위 — beatFraction

**1박 = `dur: { n: 1, d: 1 }`** (박 기준 분수, 부동소수점 없음)

| dur | 음표 | 설명 |
|-----|------|------|
| `{ n: 1, d: 4 }` | 16분음표 | 1박의 1/4 |
| `{ n: 1, d: 2 }` | 8분음표 | 1박의 1/2 |
| `{ n: 1, d: 1 }` | 4분음표 | 1박 |
| `{ n: 2, d: 1 }` | 2분음표 | 2박 |
| `{ n: 4, d: 1 }` | 온음표 | 4박 (4/4 한 마디) |

4/4 기본 마디: `[{dur:1/1}, {dur:1/1}, {dur:1/1}, {dur:1/1}]` — 합 = `beatCount`(4)

8분 분할 1박: `[{dur:1/2}, {dur:1/2}, …]`

## Song (version 3)

```json
{
  "version": 3,
  "tempo": 100,
  "timeSignature": { "num": 4, "den": 4 },
  "tracks": []
}
```

- version 2 저장본은 불러올 때 v3로 자동 변환 (`lengthUnits / beatUnits` → dur 분수)
- version 2·3 이외는 기본곡으로 대체

## Track

```json
{
  "id": "piano",
  "type": "melodic",
  "instrument": "piano",
  "measures": []
}
```

| type | measure 구조 |
|------|-------------|
| `melodic` | `{ lanes: { [pitch]: LaneCell[] } }` |
| `percussion` | `{ cells: PercussionCell[] }` |

## Melody Measure (피아노롤 레인)

```json
{
  "lanes": {
    "C4": [
      { "dur": { "n": 1, "d": 2 }, "on": true },
      { "dur": { "n": 1, "d": 2 }, "on": true },
      { "dur": { "n": 1, "d": 1 }, "on": false },
      { "dur": { "n": 1, "d": 1 }, "on": false }
    ]
  }
}
```

- 음역: **C3 ~ C5** (25개 반음)
- 각 pitch lane의 dur 합 = `beatCount`
- `on: true`이면 해당 시간에 그 음이 연주됨

## LaneCell / PercussionCell

```json
{ "dur": { "n": 1, "d": 1 }, "on": false }
{ "dur": { "n": 1, "d": 1 }, "notes": ["hit"] }
```

## 분할 / 병합

- **분할**: `dur` 분모를 2배 (1/1 → 1/2+1/2) 또는 분자를 절반 (2/1 → 1/1+1/1)
- **병합**: 인접 칸 dur 합산 (합 ≤ beatCount)
- 멜로디: on이 하나라도 true이면 병합 결과 on
- 타악기: 첫 칸 notes 유지

## 아키텍처: 노트 I/O ↔ 연주 분리

| 레이어 | 파일 | 역할 |
|--------|------|------|
| 상수 | `js/model.js` | TIME_SIGNATURES, PITCHES, INSTRUMENTS |
| 노트 I/O | `js/note-data.js` | v3 CRUD, split/merge, setTimeSignature, serialize |
| 스케줄 | `js/schedule.js` | dur + tempo + timeSignature → 절대 초 이벤트 |
| 연주 | `js/audio.js` | Tone.js Transport, 하이라이트 |

## 박자 변경 마이그레이션

트랙 전체의 칸(dur)을 **시간순으로 이어 붙인 뒤**, 새 `beatCount`로 마디를 재포장한다.

- **dur 값은 변하지 않음** — `[1/2, 1/2]`는 박자 변경 후에도 `[1/2, 1/2]`
- 마디 경계를 넘는 칸만 dur split
- 마디 수 = `ceil(총 박 수 / beatCount)`

**예: 3/4 마디 2개**

```text
M1: [도, 도/레, 레]  M2: [-, 미, -]
→ 선형 dur: [1, 1/2+1/2, 1, 1, 1, 1] …
→ 4/4: M1 [도, 도/레, 레, -], M2 [미, -, -, -]
```

## v2 → v3 변환

```
dur = lengthUnits / beatUnits  (기약 분수)
```

예: 4/4에서 lengthUnits=2 → dur=1/2 (8분), lengthUnits=8 → dur=2/1 (2분)

## 직렬화

- **localStorage**: 키 `mukebox-song` (구 `web-music-factory-song` 자동 마이그레이션)
- **파일**: `.json` (version 3)
- **URL 공유**: `#song=<base64url(JSON)>`

## 연주 스케줄링

- `schedule.js`: 각 칸 시작 = 마디 시작 + (이전 dur 합) × 박 길이(초)
- 박 길이 = `(60/tempo) × (4/den) × (num/beatCount)` … = 마디 길이 / beatCount
- `audio.js`: schedule 이벤트만 Tone.js에 전달
